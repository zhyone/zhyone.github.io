解决Cannot read property forEach of undefined错误

发表于 2017-12-21 | | 阅读 次

今天在执行hexo g的时候，意外的出现了Cannot read property 'foreach' of undefined错误，大致意思是foreach属性没有被定义。奇怪了，之前运行都正常的，网上找了很多资料，有的说是node.js版本需要更新，我把node.js更新到了最新的8.9.3版本，并且重新安装了hexo，还是没有得到解决。

更奇怪的是，我家里的电脑运行代码正常，单位的电脑运行就出现这样的错误，代码都是一样的。于是，我在家里的电脑上一步一步做测试，最终发现了问题的根源所在，原来是因为我虽然通过hexo new page about创建了about页面，并且指定了about的layout为about，但是并没有在模板里面新建about.ejs文件，才导致了这个错误。

但这样的错误提示，确实让人一头雾水。做个笔记记录一下。

解决办法：看看hexo主题目录下，是否有对应的ejs文件，如果没有，需要新建。
