将hexo同时部署到coding和github上

发表于 2017-11-15

将hexo博客同时部署到coding.net和github.com上，并且设置dns，让国内用户访问coding站点，让国外用户访问github站点。
coding.net对国内用户访问速度快，特别是百度蜘蛛比较友好，而github屏蔽了百度蜘蛛的抓取，对google蜘蛛抓取比较友好，对国外用户访问速度快。

注册github账户

目前博客已经部署在coding上，需先注册个github用户zhyone，并且新建一个名为zhyone.github.io的Repository
这里特别注意，在新建Repository的名称必须是注册用户名一致，并且Repository名称必须是用户名.github.io

否则会在部署的时候，出现以下错误，找不到远程仓库。

添加公钥到github

在settings里，找到SSH and GPG keys，点击new SSH key，将本地的公钥里的内容粘贴到里面并保存。
