通过gulp压缩去除html多余空白行

发表于 2017-11-12 | | 阅读 次

默认情况下，使用hexo的next主题，页面会出现很多的空白行，可以通过gulp来对html压缩，去除这些空白行，同时gulp还能压缩css，图片等文件。减少网站整体的尺寸大小。

如下图

解决方法：

安装cnpm

用cnpm来安装gulp能避免因网速慢下载文件不全而导致的很多问题。

安装gulp

安装gulp需要全局安装一次，再本地环境安装一次，参见gulp入门指南

安装gulp相关插件

需要用到以下插件

del
gulp-clean-css
gulp-htmlclean
gulp-htmlmin
gulp-imagemin
gulp-uglify
run-sequence

也可以写成一行
