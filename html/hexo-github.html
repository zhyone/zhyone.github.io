在github上搭建hexo博客

发表于 2017-11-19 | | 阅读 次

hexo有很多优点，除了免费，还有一点我比较看重的，就是安全，因为没有用数据库，不用担心sql注入。特别适合小型网站建设，以及个人博客搭建。以下是我在github上搭建hexo的一些笔记。供新人参考。

注册github

1)注册github账号

github网址：https://github.com/
输入你的用户名，电子邮箱，密码，然后点击sign up for GitHub注册

2)新建project项目

点击start a project

填写项目仓库名，格式必须是你的用户名.github.io,默认是public，免费的，private是收费的。点击create repository

这样就创建好了一个github仓库。

安装本地环境

1)安装Node.js

Node.js下载地址 https://nodejs.org/zh-cn/，安装很简单，默认安装即可。

2)安装Git

Git下载地址 https://git-scm.com/,选择默认安装。

本地安装hexo

先在本地磁盘新建一个目录，比如在D盘新建zhyone目录，进入zhyone目录，右键鼠标，选择右键菜单“Git bash here”，打开Git命令行窗口。

为了加快安装下载速度，我们采用cnpm用淘宝镜像来安装，先安装cnpm

安装Hexo，可以安装官方的步骤https://hexo.io/

在浏览器地址栏输入http://localhost:4000/进行访问，如果能正常显示hexo默认首页，表示Hexo在本地安装成功。
