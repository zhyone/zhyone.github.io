用git-ftp上传代码到虚拟空间

发表于 2017-11-20 | | 阅读 次

一般情况下，我们购买的虚拟空间是不支持git上传代码的，支持ftp。如果每次更新了网站代码，要再全部重新上传一遍，非常浪费时间。
有个不错的工具git-ftp，可以用git的方式来上传代码到虚拟空间。

安装git-ftp

如果是windows 10系统，c盘没有写入权限，可以下载到当前目录下

然后再将git-ftp文件拷贝到C:\Program Files\Git\mingw64\bin下

初始化git

进入本地站点根目录，初始化git

此命令会在当前目录下生成一个.git隐藏目录。

设置git环境

通过cat .git/config可以查看配置内容

为了安全起见，你也可以不设置git-ftp密码，在上传和更新代码时再输入
